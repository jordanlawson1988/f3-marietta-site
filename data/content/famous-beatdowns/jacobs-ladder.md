# Jacob's Ladder

Pair exercises at the top and bottom of a hill, increasing reps each round. Run the hill between each station.

## Structure
- Round 1: 1 Merkin (at top), run down, 1 Squat (at bottom), run back up
- Round 2: 2 Merkins, run down, 2 Squats, run back up
- Continue up to 10 Merkins and 10 Squats
- Each round includes a hill run between exercises

## Variations
- Pair burpees + squats instead of merkins + squats
- Partner version: one partner runs while the other completes reps, then switch
- Add pull-ups at the top if a pull-up bar is available
- Reduce to 5 reps at the peak for a quicker version

## Marietta notes
The back hill at The Battlefield is the standard venue. Works as a leg-focused strength and cardio hybrid.
